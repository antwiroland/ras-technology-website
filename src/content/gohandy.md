# GoHandy - Artisan Emergency Ordering Platform

## Project Overview
GoHandy is a mobile-first artisan ordering platform for the Ghanaian market. It connects customers who need urgent services with verified artisans in real time.

## Problem Statement
- Slow and unreliable access to trusted artisans
- Unverified or unskilled workers
- No structured reputation or review system
- Price and trust disputes

## Solution
GoHandy provides a trusted, location-aware marketplace that:
- Connects customers to nearby verified artisans
- Uses Ghana Card verification to prevent fraud
- Enables fast emergency order acceptance
- Builds artisan reputation through ratings and Expert Points
- Monetizes with a per-order fee

## Core User Flow
- Registration and verification
- Emergency order placement and broadcast
- Order acceptance with time limits
- Job completion and rating

## MVP Features
- Phone-based authentication with OTP
- Ghana Card backed verification
- Real-time emergency notifications
- Location-based targeting
- Wallet system with per-order deductions
